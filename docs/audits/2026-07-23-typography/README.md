# BoxSofa homepage and product typography audit

## Scope

- Homepage and Chameleon product page
- Desktop browser viewport and mobile 390 x 844
- Type hierarchy, density, repeated emphasis and responsive reading order

## Step results

1. **Homepage hierarchy - Improved**
   - Reduced the hero title from a 56 px ceiling to 50 px.
   - Standardized the three supporting paragraphs instead of giving each a competing bold style.
   - Shortened the kicker and headline across all five languages.
   - Reduced hero height so the next section is visible on desktop.

2. **Product summary - Improved**
   - Reduced the product title ceiling from 40 px to 36 px.
   - Added a quiet breadcrumb and regular-weight description.
   - Removed the repeated shipping and payment paragraph.

3. **Trust and buying controls - Improved**
   - Replaced four bordered trust cards with one lightweight divided list.
   - Kept price and primary actions as the strongest conversion elements.
   - Reduced option-label and general button weight.

4. **Mobile reading order - Healthy**
   - Hero title is 29 px with a less aggressive line height.
   - Product title is 23 px and trust details remain readable in two columns.
   - No text overflow was observed at 390 px.

5. **Accessibility evidence - Limited pass**
   - Text remains visible at desktop and mobile breakpoints.
   - Color contrast appears strong in captured states.
   - Keyboard navigation, zoom at 200%, and screen-reader pronunciation require separate testing.
